Sobre FitTracker IoT

Sistema de monitoreo de actividad física que simula un dispositivo wearable inteligente (smartwatch/pulsera fitness) utilizando tecnología ESP32 y almacenamiento en la nube.

Tecnologías Utilizadas

Flask (Python)
Framework web backend
MongoDB Atlas
Base de datos NoSQL en la nube
Bootstrap 5
Framework CSS para diseño responsive
Docker
Containerización de aplicaciones
Chart.js
Visualización de datos en tiempo real
ESP32
Microcontrolador IoT (simulado)

Sensores Biométricos Implementados

Sensor de Frecuencia Cardíaca
Tipo: Óptico (PPG)

Monitoreo continuo del ritmo cardíaco mediante fotopletismografía.

Rango: 60-180 bpm | Zonas: Reposo, Ligera, Moderada, Intensa, Máxima
Contador de Pasos
Tipo: Acelerómetro

Detección de pasos mediante análisis de movimiento triaxial.

Meta diaria: 10,000 pasos | Precisión: ±2%
Oxímetro de Pulso (SpO₂)
Tipo: Espectroscopia

Medición no invasiva de saturación de oxígeno en sangre.

Rango normal: 95-100% | Alerta: < 90%
Sensor de Temperatura Corporal
Tipo: Termistor NTC

Monitoreo de temperatura de piel durante la actividad física.

Rango normal: 36.1-37.5°C | Precisión: ±0.1°C

Características del Sistema

  • Monitoreo en tiempo real
  • Gráficas históricas interactivas
  • Zonas de entrenamiento cardíaco
  • Contador de calorías estimadas
  • Cálculo de distancia recorrida
  • API REST para integraciones
  • Diseño responsive (móvil/tablet/desktop)
  • Almacenamiento en la nube
  • Alertas de niveles anormales
  • Fácil migración a hardware real

Aplicaciones Prácticas
  • Deportistas: Monitoreo de rendimiento y zonas de entrenamiento
  • Salud preventiva: Detección temprana de anomalías cardiovasculares
  • Rehabilitación: Seguimiento de progreso en terapias físicas
  • Investigación: Recolección de datos biométricos para estudios
Migración a Hardware Real

Este proyecto está diseñado para facilitar la implementación con hardware real. Componentes necesarios: ESP32, sensor MAX30102 (FC + SpO₂), MPU6050 (acelerómetro), y sensor MLX90614 (temperatura IR).
